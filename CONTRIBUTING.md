# Contributing to Injecticide

Injecticide is licensed under GPLv3. Any code contributed to this project
must also be released under GPLv3. When contributing, you agree that your
contributions become part of the GPLv3-licensed project.

## How to Contribute

1. Fork the repository.
2. Create a feature branch describing your change:
   `git checkout -b feature/my-change`
3. Write clean, documented Python code.
4. Add or update tests under `tests/`.
5. Ensure everything passes:
   `pytest`
6. Submit a Pull Request with a clear description.

## Code Guidelines

- Python 3.10 or higher.
- Flat project layout unless complexity requires modules.
- Keep functions small, focused, and testable.
- No proprietary dependencies.

## Reporting Issues

Use GitHub Issues for bug reports, feature requests, or design proposals.
